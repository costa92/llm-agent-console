| replayed/persisted event payloads → DOM | Untrusted persisted event strings rendered into the replay timeline |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-V5 | Tampering / Elevation | RunDetail + replay TimelineView | mitigate | replayed payloads (persisted node outputs/errors) rendered as React TEXT nodes via the same TimelineView/RawJsonViewer — never dangerouslySetInnerHTML (RESEARCH V5) |

| T-03-11 | Availability/correctness | empty /events handling | mitigate | 200+empty events → empty state, NOT an error (Pitfall 7) — asserted in RunDetail.test |
