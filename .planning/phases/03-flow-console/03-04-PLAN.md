|----------|-------------|
| console → BFF `/api/flow/.../run/stream` + `/run` | NO bearer / NO X-Console-* (BFF injects) |
| streamed event payloads (LLM/tool output) → DOM | Arbitrary upstream strings rendered into the live timeline |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-V5 | Tampering / Elevation | TimelineView / NodeStatusList / RunResultPanel | mitigate | render ALL flowd event-payload strings (output/error/metadata/node names) as React TEXT nodes; raw-JSON viewer shows escaped text; NEVER dangerouslySetInnerHTML — heightened here because payloads stream from arbitrary node outputs (RESEARCH V5). Asserted by TimelineView.test rendering a payload with markup chars and checking it is escaped. |
| T-03-08 | Spoofing / confused-deputy | run/stream + run calls | mitigate | RunTrigger/useSyncRun send NO Authorization / X-Console-* (the 03-01 client/stream already omit them); no auth header added at the call site |
| T-03-06 | DoS / resource | streamed run lifecycle | mitigate | useRunStream (Plan 03) aborts on terminal + unmount; this UI mounts/unmounts the hook so navigating away aborts the run (flowd r.Context() cancels) |
