| Boundary | Description |
|----------|-------------|
| Operator context (unset) → recall | D-12 gate blocks all recall calls until tenant+user set; prevents 401 doomed requests |
| Gateway recall strings (content/tags) → DOM | Untrusted memory content rendered into the table; React text-node escaping is the XSS boundary |
| URL search params → recall request | query/top_k from the URL drive the POST; top_k clamped ≤ 50 to avoid gateway 400 |
