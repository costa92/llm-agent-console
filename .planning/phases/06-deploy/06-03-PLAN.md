    ## Nginx Proxy Settings (SC-3: Required LB/Proxy Configuration)
    Include a table with ALL of the following rows (values from the actual nginx.conf):

    | Setting | Value | Location in nginx.conf | Notes |
    |---------|-------|------------------------|-------|
    | proxy_buffering (SSE) | off | SSE location block | MANDATORY — enables per-event flush |
    | proxy_cache (SSE) | off | SSE location block | Defense-in-depth |
    | gzip (global) | off | global directive | Belt-and-braces: prevents accidental re-enable |
    | gzip (SSE block) | off | SSE location block | Primary control |
    | proxy_read_timeout (SSE) | 3600s (1 hour) | SSE location block | Covers silent LLM steps >60s |
    | proxy_send_timeout (SSE) | 3600s | SSE location block | Paired with read timeout |
    | proxy_read_timeout (REST) | 60s | /api/ location block | Standard for short REST calls |
    | proxy_http_version (SSE) | 1.1 | SSE location block | Required for chunked streaming |
    | Connection header (SSE) | '' (empty) | SSE location block | Plain HTTP/1.1 SSE, NOT WebSocket |
    | X-Accel-Buffering | passed through | SSE location block | BFF sets 'no'; nginx relays it |

    Also include the SSE path coverage table showing all 4 matched paths:

    | SSE Path | nginx regex match | Notes |
    |----------|-------------------|-------|
    | /api/stream/test | YES (.*stream) | Synthetic proof endpoint |
    | /api/flow/*/run/stream | YES (.*stream) | Live flowd run stream |
    | /api/flow/runs/*/replay | YES (.*replay) | flowd replay SSE (GAP-1 fix) |
    | /api/chat/stream | YES (.*stream) | Customer-support chat stream |

    Note: "There is NO upstream load balancer or reverse proxy in front of this nginx. The
    console's nginx IS the edge proxy. If a second proxy (e.g. Caddy for TLS) is added in
    the future, the settings in the table above MUST be replicated on that proxy for SSE routes."
