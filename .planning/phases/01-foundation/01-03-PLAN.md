<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → BFF (app layer) | Operator token in Authorization: Bearer header; checked constant-time before route dispatch |
| BFF Rewrite hook → upstream memory-gateway | BFF enforces header strip+re-materialize; no client-controlled scope headers reach upstream |
| BFF Rewrite hook → upstream flowd | BFF injects flowd bearer token from config; client Authorization is stripped before injection |
| BFF → /api/config/env response | Response must not include flowd_token or operator_token fields |
| BFF ModifyResponse → client (flowd path) | ModifyResponse scrubs any echo of the flowd token from upstream response headers before forwarding to client |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Information Disclosure | flowd bearer token in BFF responses | mitigate [BLOCKING] | flowd_token exists only in cfg.FlowdToken; the /api/config/env handler explicitly excludes it; ModifyResponse deletes Authorization + X-Echo-Auth from upstream response before forwarding; TestFlowDirectorResponseNoToken asserts the token value is absent from all client-facing response headers and body |
| T-03-02 | Spoofing (confused deputy) | Client-sent X-Tenant-Id/X-User-Id bypassing re-materialization | mitigate [BLOCKING] | Rewrite hook calls Del on ALL X-*-Id headers BEFORE calling Set from X-Console-*; order is mandatory — test explicitly verifies that a client-sent X-Tenant-Id value does NOT appear in the outbound request when X-Console-Tenant has a different value |
| T-03-03 | Elevation of Privilege | Client-sent Authorization forwarded to flowd | mitigate [BLOCKING] | All three directors call r.Out.Header.Del("Authorization") FIRST, then conditionally Set the service-specific token; the flow director's test asserts the flowd token (not the operator token) is in the outbound Authorization header |
| T-03-04 | Tampering | SSRF via client-controlled upstream URL | mitigate [BLOCKING] | r.SetURL(cfg.MemoryBase/FlowBase/ChatBase) uses only the config-file-pinned base URLs; no URL or host from the inbound request is used; the allowlist (http.StripPrefix + mux pattern) rejects any path not under /api/{memory,flow,chat}/* |
| T-03-05 | Spoofing | Operator token timing attack | mitigate | Constant-time byte-XOR comparison per flowd BearerTokenAuthenticator pattern; length check before XOR loop (rejects different-length tokens without XOR, avoiding padding oracle) |
| T-03-SC | Tampering | gopkg.in/yaml.v3 package install | mitigate | Package present in Go module cache; verified HIGH confidence in RESEARCH.md sources; no postinstall script |
</threat_model>
